Contexto:
Estou rodando um projeto Node.js + Express com Vite no Replit chamado “EcoTrackSolution”. 
O front-end em React (Vite) faz uma chamada POST para /api/businesses ao submeter o formulário de “Adicionar Novo Negócio”. 
O servidor Express não possui essa rota, então qualquer POST para /api/businesses cai no fallback do Vite e serve index.html (HTML). 
O front-end faz response.json() e lança “Unexpected token '<', '<!DOCTYPE '… is not valid JSON”.

Objetivo:
1. Adicionar no servidor Express uma rota POST /api/businesses que:
   - Valide o corpo da requisição com Zod usando o esquema insertBusinessSchema (importado de "@shared/schema").
   - Chame storage.createBusiness(validatedData) para inserir no banco (ou storage in-memory).
   - Em caso de sucesso, retorne status 201 e o JSON do novo registro.
   - Em caso de ZodError, retorne status 400 com { message: "Validation error", errors: [...] }.
   - Em caso de qualquer outro erro, retorne status 500 com { message: "Failed to create business" }.

2. Certificar que o middleware express.json() está ativo (já está configurado corretamente no index.ts).

3. Garantir que a rota POST /api/businesses seja registrada **antes** do middleware de fallback do Vite (setupVite/serveStatic), dentro de registerRoutes(app) ou imediatamente após.

4. Importar no topo de server/routes.ts:
   ```ts
   import { insertBusinessSchema } from "@shared/schema";
   import { z } from "zod";
Validar que não existam conflitos de import ou duplicações de rotas.

Após inserir o código, reiniciar o servidor pelo processo padrão do Replit (botão ‘Restart’ ou npm script).

Detalhes técnicos:

A função storage.createBusiness já está implementada em server/storage.ts, retornando o novo objeto Business.

O arquivo server/index.ts aplica express.json(), CORS e logging antes de chamar registerRoutes(app).

As demais rotas GET, PATCH e DELETE para /api/businesses já estão funcionando, portanto a única ausência é o POST.

Use sintaxe TypeScript compatível com o restante do projeto.

O ponto exato de inserção é antes de if (app.get("env") === "development") { await setupVite(app, server); }.
