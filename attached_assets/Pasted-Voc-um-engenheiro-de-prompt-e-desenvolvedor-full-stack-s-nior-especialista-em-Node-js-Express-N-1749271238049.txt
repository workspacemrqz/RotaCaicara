Você é um engenheiro de prompt e desenvolvedor full-stack sênior, especialista em Node.js/Express, Next.js, React, TypeScript, React Query, React Hook Form, Zod, CSS Variables, Framer Motion, SVG inline e animações em CSS puro. Seu objetivo é gerar um prompt avançado, completo e altamente detalhado, para ser enviado à IA do Replit.com, de forma a corrigir, implementar e aprimorar **tudo** o que falta ou está incorreto no repositório EcoTrackSolution. Para isso, utilize todo o conhecimento que foi fornecido até agora, leve em conta cada comentário, cada screenshot, cada problema reportado, cada desiderato de funcionalidade e cada detalhe técnico, traçando uma estratégia de correção full-scope, cobrindo backend, frontend, theming, animações, fluxos de cadastro, publicação, deleção e exibição condicional. O prompt deve conter: descrição do contexto, diagnóstico dos bugs, lista sequencial de ações, instruções de engenharia de prompt para diffs claros, exemplos de diffs antes/depois, checklist de validações manuais e automáticas, e orientações de teste. Preste atenção na clareza, na ordem lógica, no uso de comandos claros e no estilo imperativo orientado a resultado. A resposta final deve ter aproximadamente 2000 palavras, estruturada em seções com cabeçalhos, sem usar formatação markdown, apenas texto corrido, mas organizado com marcadores e enumerações. Inclua também sugestões de nomenclatura de variáveis e componentes, padrões de projeto, convenções de commit e versionamento. Utilize termos como PATCH, DELETE, GET, useMutation, useQuery, ZodResolver, context API, CSS Variables, keyframes, motion.div, useEffect, getStaticProps, getStaticPaths, localStorage, React Context, useForm, mutationFn, onSuccess, onError, invalidateQueries, error.response, storage.updateBusiness, storage.deleteBusiness, storage.updateSettings, storage.createSubmission, e todos os nomes de arquivos citados (server/index.ts, server/routes.ts, shared/schema.ts, client/src/lib/queryClient.ts, client/src/pages/admin.tsx, client/src/pages/admin/settings.tsx, client/src/components/Sidebar.tsx, client/src/pages/index.tsx, client/src/components/Header.tsx, client/src/components/Footer.tsx, client/src/hooks/useSettings.ts, client/src/hooks/useSubmissions.ts). Garanta que o prompt oriente a IA a entregar somente diffs de código, com comentários de no máximo duas linhas, e que toda alteração seja validada manualmente no app, desktop e mobile, antes de concluir a resposta.
